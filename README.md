# SEEK — Seek and You Shall Find

An immersive ARG (Alternate Reality Game) that blends physical and digital experiences into a mystery-driven treasure hunt.

Players are challenged to solve puzzles, decode hidden messages, and explore real-world environments while interacting with a web-based system.

## 🧠 Concept

"Seek and you shall find."

This project is designed to test observation, logic, and curiosity. Each step reveals part of a hidden narrative, where players must connect clues across different layers — physical objects, encoded messages, and digital interactions.

## 🎮 Features

- 🔍 Puzzle-solving and cryptography elements
- 🌐 Web-based progression system
- 📱 QR code integration
- 🧩 Multi-layer challenges (physical + digital)
- 🕵️ Narrative-driven experience (ARG style)

## 🚀 Tech Stack

- HTML, CSS, JavaScript
- (Future) Backend for player tracking
- (Future) Database for progression system

## 🧪 Status

🚧 In development — Phase 1 prototype

## 🎯 Goal

To create an engaging, scalable treasure hunt experience that can be played in different environments (home, events, outdoor spaces).
